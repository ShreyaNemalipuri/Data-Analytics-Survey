
Data Analytics Survey Dashboard is an interactive Power BI project designed to analyze survey responses collected from participants across different demographics and professional backgrounds.
This dashboard transforms raw survey data into meaningful insights using powerful data modeling, cleaning, and visualization techniques.

The project highlights:

Trends in skill levels across visualization, programming, and database knowledge

Tool preferences among data professionals

Course preferences, learning goals, and satisfaction levels

Demographic distribution of respondents

Comparative analysis between student and working professionals

This dashboard serves as a useful resource for institutions, educators, and organizations to understand learning trends in the data analytics domain and make informed decisions based on visual insights.
